Глосарій показників

Назви метрик такі, як у вихідних CSV; нижче — коротке пояснення українською для звіту з реклами Sponsored listings.

Provider ID
Унікальний ідентифікатор закладу (точки) у системі.
Provider Name
Назва локації партнера, як відображається в даних.
City ID
Внутрішній код міста; у цьому звіті 158 відповідає Києву, 496 — Львову (узгоджено з бренд-групою LVIV).
Country Name
Країна локації (у даних — Ukraine).
Business Segment
Бізнес-сегмент партнера; для Муза в файлах — Mid-market (AM Segment).
Start / End Date (Local)
Початок і кінець періоду звітування в локальному часі.
Placement
Місце показу оголошення; у даних — home_screen (головний екран застосунку).
State
Статус кампанії / розміщення; у даних — active (активне).
Total Spend (EUR)
Загальні витрати на рекламу в євро за період.
ROAS
Return on Ad Spend — віддача від рекламних витрат; відношення атрибутованого доходу до витрат на рекламу. Чим вище, тим ефективніше окупаються витрати.
Orders (Attributed)
Кількість замовлень, приписаних (атрибутованих) цьому рекламному показу за обраною моделлю атрибуції.
Attributed NET Revenue
Чистий дохід від атрибутованих замовлень (після віднімань), у євро.
Provider Net Revenue (Attributed, EUR)
Аналог доходу на рівні групи бренду: чистий атрибутований дохід провайдера в євро.
Impressions
Покази — скільки разів користувачі бачили рекламний банер / картку.
Clicks
Кліки — скільки разів користувачі натиснули на рекламне оголошення.
CPM
Cost Per Mille — вартість тисячі показів (євро за 1000 impressions).
CPC
Cost Per Click — середня вартість одного кліку.
Cost per order
Середня вартість реклами на одне атрибутоване замовлення (витрати / замовлення).
New Users (Attributed)
Нові користувачі, атрибутовані до цієї реклами (перше замовлення або реєстрація — залежно від визначення у продукті Bolt).
Cost per new user
Середня вартість залучення одного нового атрибутованого користувача.
CTR %
Click-Through Rate — частка кліків від показів (кліки / покази × 100%). Показує привабливість креативу в стрічці.
CVR %
Conversion Rate — частка замовлень від кліків (замовлення / кліки × 100%). Показує, наскільки клік перетворюється на замовлення.
Brand Name / Group Name
Назва бренду та об’єднання точок у звіті групи (наприклад, MUZA GROUP).
